# 💻 C-Programming-Practice

![C](https://img.shields.io/badge/Language-C-blue)
![Status](https://img.shields.io/badge/Status-Active-success)

C-Programming-Practice is a collection of classic and advanced C programming examples, including basic algorithms, data structures, and number manipulations.  

## 🚀 Features

### C-Temel-Ornekler (Beginner)
- Faktöriyel ve Fibonacci
- Asal sayı kontrolü
- Toplama, Çıkarma, Çarpma, Bölme
- Dizi ortalaması ve maksimum-minimum bulma
- Basit hesap makinesi
- Tek/Çift sayı kontrolü
- Basit sıralama (Bubble Sort)

### C-İleri-Ornekler (Intermediate/Advanced)
- Stack ve Queue (dizi ve döngüsel)
- Tek yönlü bağlı liste ve tersine çevirme
- Merge Sort
- Tail Recursive Faktöriyel
- Matris transpozları
- Binary ↔ Decimal dönüşümleri
- Palindrom sayı kontrolü
- Asal sayılar belirli aralıkta
- EBOB/EKOK hesaplamaları
- Dizi büyükten küçüğe sıralama  

## 🛠 Technologies Used
- C (ANSI C standard)
- Standard C libraries (`stdio.h`, `stdlib.h`, `math.h`)  

## 📌 Project Status
This project is actively maintained. Examples are tested and ready to run.
